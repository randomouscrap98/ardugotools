# Ardugotools

A CLI toolset for Arduboy

## Features

- Scan for connected devices
- Analyze connected devices
- Read sketch, eeprom, ~flashcart~
- Write ~sketch~, eeprom, ~flashcart~
- Analyze flashcart
- Convert between various data formats

(More to come)

## Building / Using 

```
go build
./ardugotools --help
```
